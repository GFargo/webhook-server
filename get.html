Tingbot webhook channel {channel}

This is the webhook channel {channel}. You can POST to this URL to change the value, and subscribe to updates on Tingbot using the @webhook decorator.

Current value

{channel_value}

Example code

import tingbot
from tingbot import *

screen.fill(color='black')
screen.text('Waiting...')

@webhook('{channel}')
def on_webhook(data):
    screen.fill(color='black')
    screen.text(data, font_size=15, color='blue')

tingbot.run()

For more information, see the webhooks section of the Tingbot documentation
